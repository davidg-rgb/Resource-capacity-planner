  <action>
    1. Create `src/features/platform/platform-impersonation.service.ts` (per D-12, D-13, D-15):
       - Import `clerkClient` from `@clerk/nextjs/server`, `db` from `@/db`, `impersonationSessions` from `@/db/schema`, `env` from `@/lib/env`, `eq, and, isNull` from `drizzle-orm`.
       - Import `createHash` from `crypto` for token hashing.

       - Export `startImpersonation(adminId: string, targetUserId: string, targetOrgId: string)`:
         - Calculate expiresAt: `new Date(Date.now() + env.IMPERSONATION_MAX_DURATION_MINUTES * 60 * 1000)`.
         - Call Clerk: `const client = await clerkClient(); const actorToken = await client.actorTokens.createActorToken({ userId: targetUserId, actor: { sub: 'platform-admin:' + adminId }, expiresInSeconds: env.IMPERSONATION_MAX_DURATION_MINUTES * 60 })`.
         - Hash token for storage: `const tokenHash = createHash('sha256').update(actorToken.token).digest('hex')`.
         - Insert into `impersonationSessions`: `{ adminId, targetOrgId, targetUserId, tokenHash, expiresAt }`.
         - Return `{ sessionId: session.id, url: actorToken.url }`.

       - Export `endImpersonation(sessionId: string, adminId: string)`:
         - Update `impersonationSessions` set `endedAt = new Date()` where id=sessionId and adminId=adminId and endedAt is null.
         - Return success boolean.

       - Export `listActiveSessions(adminId?: string)`:
         - Select from impersonationSessions where endedAt is null and expiresAt > now(). Join with platformAdmins for admin name. Optionally filter by adminId.
         - Return array of active sessions with admin name, target org/user info.

    2. Create `src/features/platform/platform-user.service.ts` (per D-19):
       - Import `clerkClient` from `@clerk/nextjs/server`.

       - Export `searchUsers(query: string, limit = 20)`:
         - Call `const client = await clerkClient(); const users = await client.users.getUserList({ query, limit })`.
         - Map results to `{ id: user.id, email: user.primaryEmailAddress?.emailAddress, firstName: user.firstName, lastName: user.lastName, organizationMemberships: user.organizationMemberships }`.
         - Return the mapped list.

       - Export `resetUserPassword(userId: string, newPassword: string)`:
         - Call `const client = await clerkClient(); await client.users.updateUser(userId, { password: newPassword })`.
         - Return `{ success: true }`.

       - Export `forceLogoutUser(userId: string)`:
         - Call `const client = await clerkClient(); const sessions = await client.sessions.getSessionList({ userId })`.
         - For each active session: `await client.sessions.revokeSession(session.id)`.
         - Return `{ revokedCount: sessions.data.length }`.

    3. Create `src/features/platform/platform-audit.service.ts` (per D-17):
       - Import `db` from `@/db`, `platformAuditLog, platformAdmins` from `@/db/schema`, `eq, and, gte, lte, desc, sql, count` from `drizzle-orm`.

       - Export interface `AuditQueryParams { adminId?: string; action?: string; startDate?: string; endDate?: string; page?: number; pageSize?: number; }`.

       - Export `queryAuditLog(params: AuditQueryParams)`:
         - Build conditions array from params (adminId filter, action filter, date range filters).
         - Select from platformAuditLog joined with platformAdmins (for admin name/email).
         - Order by createdAt desc.
         - Apply pagination: offset = (page-1) * pageSize, limit = pageSize. Default page=1, pageSize=50.
         - Also run count query with same conditions for total.
         - Return `{ entries: Array<{id, adminName, adminEmail, action, targetOrgId, targetUserId, details, ipAddress, createdAt}>, total, page, pageSize }`.
  </action>
