**Step 1: Scaffold with create-next-app**

Run from the project root directory (which IS the repo root). The app should be created IN the current directory, not a subdirectory. Use:

```bash
pnpm create next-app . --typescript --tailwind --eslint --app --turbopack --use-pnpm
```

If prompted about overwriting existing files, accept. If `create-next-app` refuses to run in a non-empty directory, create in a temp subdirectory and move files up.

**Step 2: Pin TypeScript to 5.7.x**

After scaffolding, check the installed TypeScript version. If it is 6.x, downgrade:

```bash
pnpm add -D typescript@~5.7.0
```

Verify `tsconfig.json` has `"strict": true`. If not, add it. Ensure `"moduleResolution"` is set to `"bundler"` (Next.js default). Do NOT use `"moduleResolution": "classic"`.

**Step 3: Install additional dev dependencies**

```bash
pnpm add -D prettier prettier-plugin-tailwindcss husky lint-staged
```

**Step 4: Configure Prettier**

Create `.prettierrc` with exact content:

```json
{

  "tabWidth": 2,
  "printWidth": 100,
  "plugins": ["prettier-plugin-tailwindcss"]
}
```

Create `.prettierignore` with:

```

```

**Step 5: Configure ESLint flat config**

Ensure `eslint.config.mjs` exists (create-next-app 16 may generate it). It must contain:

```javascript
import { dirname } from 'path';
import { fileURLToPath } from 'url';
import { FlatCompat } from '@eslint/eslintrc';

const __filename = fileURLToPath(import.meta.url);
const __dirname = dirname(__filename);

const compat = new FlatCompat({
  baseDirectory: __dirname,
});

const eslintConfig = [
  ...compat.extends('next/core-web-vitals', 'next/typescript'),
  {
    rules: {
      '@typescript-eslint/no-unused-vars': ['error', { argsIgnorePattern: '^_' }],
    },
  },
];

export default eslintConfig;
```
