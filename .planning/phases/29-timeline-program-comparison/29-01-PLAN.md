# Phase 29 Plan 01: Timeline + Program + Comparison Widgets (C4)

## Objective

Build 3 advanced visualization widget components (V2, V10, V11) implementing WidgetProps and registered in the widget registry.

## Context

- Spec: .planning/DASHBOARD-VISUALIZATIONS-SPEC.md (V2, V10, V11 sections)
- Widget registry: src/features/dashboard/widget-registry.ts
- Data hooks: use-availability-timeline.ts, use-program-rollup.ts, use-period-comparison.ts
- Pattern reference: heat-map-table.tsx (custom HTML table), department-bar-widget.tsx (widget wrapper)

## Tasks

### Task 1: V2 Resource Availability Timeline Widget
type="auto"

Create `availability-timeline-widget.tsx`:
- Gantt-style custom HTML table (rows = people, columns = months)
- Department collapsible groups
- Colored allocation bars per project within cells
- Available-only filter toggle
- Legend and summary insight bar
- Current month highlight

### Task 2: V10 Program Portfolio Roll-up Widget
type="auto"

Create `program-rollup-widget.tsx`:
- KPI cards (projects, people, peak load)
- SVG donut staffing completeness gauge
- Discipline coverage horizontal bars with gap alerts
- Monthly load bars with peak highlighting
- Project list with status badges
- Gap alert callout

### Task 3: V11 Period-over-Period Comparison Widget
type="auto"

Create `period-comparison-widget.tsx`:
- Quarter/month preset selectors
- Key metrics table with signal indicators (improving/worsening/neutral)
- Department shifts table sorted by absolute delta
- Notable changes bullet list
- Delta arrows with contextual coloring

### Task 4: Widget Registry Integration
type="auto"

- Add all 3 imports to widgets/index.ts
- Verify TypeScript compilation passes
