       - `'use client'` component.

       - Render a plain HTML table (NOT AG Grid -- read-only view per research anti-pattern guidance) with Tailwind styling:
         - Header row: first column "Person", then one column per month (formatted as "Mar 2026" short format using `new Date(month + '-01').toLocaleDateString('en', { month: 'short', year: 'numeric' })`).
         - One row per person: person name as a `<Link href={/input/${person.personId}} className="text-primary hover:underline font-medium">` (per PROJ-03), then hours for each month from `person.months[month] ?? 0`.
         - If hours for a month are 0, render with `text-on-surface-variant/50` muted styling.
         - If hours > 0, render with `text-on-surface font-medium`.
       - If `people.length === 0`, render empty state: "No people allocated to this project. Go to the Input page to add allocations."
       - Table styling: `w-full text-left text-sm`, header cells with `font-headline text-on-surface-variant`, body with `divide-outline-variant divide-y`, horizontal overflow with `overflow-x-auto` wrapper.

    2. **Create `src/components/project-view/project-summary-row.tsx`** (per PROJ-02, PROJ-04):
       - `'use client'` component.
       - Props: `people: ProjectStaffingPerson[]`, `months: string[]`.
       - Compute totals: for each month, sum `person.months[month] ?? 0` across all people using `Array.reduce`.
       - Render a table row (`<tr>`) meant to be appended inside the same `<tbody>` as the grid (or as a `<tfoot>`):
         - First cell: "Total" in bold.
         - One cell per month: total hours for that month.
         - **Understaffed indicator (per PROJ-04):** If total hours for a month is 0 AND there is at least one person allocated to the project (i.e., `people.length > 0`), style the cell with `bg-amber-100 border-dashed border-amber-300 border` and show "--" text with a title tooltip "No allocations this month". This visually flags months where the project has team members but zero hours allocated.
         - If total is 0 and no people at all, just show "0" with muted styling (no amber).
         - Normal cells with hours > 0: `font-semibold text-on-surface`.
       - Summary row has `bg-surface-container-low` background to visually distinguish from person rows.
       - Export as named `ProjectSummaryRow`.
