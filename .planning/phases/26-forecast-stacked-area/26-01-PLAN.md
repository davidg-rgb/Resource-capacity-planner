# Phase 26 Plan 01: Forecast + Stacked Area Charts

## Objective

Build two Recharts-based visualization components for the dashboard framework:
- V1: Capacity Forecast Line Chart (demand vs supply with gap shading + hiring trigger)
- V5: Stacked Area Chart (hours by project/department/discipline with "Other" bucket)

## Context

- Spec: `.planning/DASHBOARD-VISUALIZATIONS-SPEC.md` sections V1 and V5
- Widget registry pattern established in Phase 23
- Data hooks created in Phase 24 (`useCapacityForecast`, `useCapacityDistribution`)
- Analytics types define `CapacityForecastResponse` and `CapacityDistributionResponse`

## Tasks

### Task 1: V1 Capacity Forecast Line Chart `type="auto"`

Create `src/components/charts/capacity-forecast-chart.tsx`:
- ComposedChart with supply (green) and demand (primary) lines
- Gap shading using range-type Area elements ([min, max] tuples)
- Green shading for surplus months, red for deficit months
- Optional hiring trigger ReferenceLine with dashed style
- Custom tooltip showing supply/demand/gap with percentage
- Summary bar below chart (surplus/balanced/deficit counts)
- Click handler for deficit months (opens availability finder)

Create `src/features/dashboard/widgets/capacity-forecast-widget.tsx`:
- Implements WidgetProps interface
- Uses useCapacityForecast hook with timeRange
- Loading/error/empty states matching existing widget pattern
- Registered as 'capacity-forecast', category 'health-capacity', full width

### Task 2: V5 Stacked Area Chart `type="auto"`

Create `src/components/charts/stacked-area-chart.tsx`:
- AreaChart with stacked areas per group (project/department/discipline)
- Dashed supply overlay line
- Interactive legend (click to toggle series visibility)
- "Other" bucket for groups beyond limit (top 7 + Other)
- Custom tooltip with per-group breakdown and percentages
- Insight callout text below chart
- Click handler for navigating to group detail

Create `src/features/dashboard/widgets/capacity-distribution-widget.tsx`:
- Implements WidgetProps interface
- Uses useCapacityDistribution hook with groupBy toggle
- Group-by toggle UI (Project/Department/Discipline)
- Registered as 'capacity-distribution', category 'breakdowns', full width

### Task 3: Widget Registration `type="auto"`

Register both widgets in `src/features/dashboard/widgets/index.ts`.

## Verification

- `npx tsc --noEmit` passes with zero errors
- Both widgets registered in widget registry index
- Components follow existing patterns (React.memo, Loader2 spinner, error states)

## Success Criteria

- V1 chart renders supply/demand lines with gap shading and hiring trigger
- V5 chart renders stacked areas with supply overlay and group-by toggle
- Both conform to WidgetProps and register correctly
- TypeScript compiles cleanly
