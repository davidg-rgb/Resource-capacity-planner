**Create `src/features/platform/platform-tenant-data.service.ts`:**

1. Export `TenantExportData` interface:
   ```typescript
   export interface TenantExportData {
     exportedAt: string;
     organizationId: string;
     organizationName: string;
     departments: Array<{ id: string; name: string; createdAt: string }>;
     disciplines: Array<{ id: string; name: string; abbreviation: string; createdAt: string }>;
     programs: Array<{ id: string; name: string; description: string | null; createdAt: string }>;
     people: Array<{ id: string; firstName: string; lastName: string; departmentId: string; disciplineId: string; targetHoursPerMonth: number; sortOrder: number; archivedAt: string | null; createdAt: string }>;
     projects: Array<{ id: string; name: string; programId: string | null; status: string; archivedAt: string | null; createdAt: string }>;
     allocations: Array<{ id: string; personId: string; projectId: string; month: string; hours: number; createdAt: string }>;
     importSessions: Array<{ id: string; userId: string; fileName: string; status: string; rowCount: number; createdAt: string }>;
   }
   ```

2. Export `async function exportTenantData(orgId: string): Promise<TenantExportData>` that:
   - Fetches the organization name from `organizations` table using `eq(organizations.id, orgId)`
   - Throws `NotFoundError` if org not found
   - Queries each tenant-scoped table (departments, disciplines, programs, people, projects, allocations, importSessions) filtered by `WHERE organization_id = orgId`
   - Does NOT include featureFlags in export (platform config, not tenant data)
   - Does NOT include parsedData/mappings/validationResult/importResult from importSessions (large JSONB blobs, not useful for export)
   - Returns the assembled `TenantExportData` object with `exportedAt: new Date().toISOString()`

3. Export `async function purgeTenantData(orgId: string): Promise<{ deletedCounts: Record<string, number> }>` that:
   - Verifies org exists (throw NotFoundError if not)
   - Deletes all tenant-scoped data in FK-safe order using `db.delete(table).where(eq(table.organizationId, orgId))`:
     1. `allocations`
     2. `importSessions`
     3. `featureFlags`
     4. `people`
     5. `projects`
     6. `programs`
     7. `disciplines`
     8. `departments`
   - Each delete uses `.returning({ id: table.id })` to count rows deleted
   - Returns `{ deletedCounts: { allocations: N, importSessions: N, ... } }`
   - Does NOT delete the organization record itself (per GDPR: purge data, keep account shell)
   - Does NOT wrap in a transaction (each delete is independent and idempotent — if one fails, retry is safe since earlier deletes already succeeded)

**Create `src/app/api/platform/tenants/[orgId]/export/route.ts`:**
- GET handler: `requirePlatformAdmin()`, extract orgId from params, call `exportTenantData(orgId)`
- Return JSON response with headers:

  ```typescript
  return new Response(JSON.stringify(data, null, 2), {

  ```
- Log audit: `logPlatformAction({ adminId: admin.adminId, action: 'tenant.export', targetOrgId: orgId })`
- Use handleApiError for errors

**Create `src/app/api/platform/tenants/[orgId]/purge/route.ts`:**
- POST handler: `requirePlatformAdmin()`, extract orgId from params
- Parse body: `{ confirmName: string }` — validate with Zod or inline check
- Fetch tenant detail using `getTenantDetail(orgId)` to get the org name
- Compare `confirmName` against `tenant.name` (exact match, case-sensitive)
- If no match: return 400 `{ error: 'Confirmation name does not match' }`
- If match: call `purgeTenantData(orgId)`, log audit with details including deletedCounts
- Return `{ success: true, deletedCounts }` with 200
- Log audit: `logPlatformAction({ adminId: admin.adminId, action: 'tenant.purge', targetOrgId: orgId, details: { deletedCounts } })`
