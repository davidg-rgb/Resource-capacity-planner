```typescript
import { drizzle } from 'drizzle-orm/neon-http';
import * as schema from './schema';
```

3. After all table definitions, add **Drizzle relations** declarations using the `relations()` API from `drizzle-orm`. Define relations for all 13 tables so the relational query builder (`db.query.people.findMany({ with: { department: true } })`) works from day one. Example:

```typescript
export const organizationsRelations = relations(organizations, ({ many }) => ({
```
