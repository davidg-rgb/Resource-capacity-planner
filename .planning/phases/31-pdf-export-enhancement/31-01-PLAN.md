# Phase 31 Plan 01: PDF Export Enhancement

## Objective

Enhance PDF export to support multi-widget dashboards with pre-export modal, SVG snapshot capture, multi-widget layout engine, single-widget export, and cover page.

## Context

- Spec: `.planning/DASHBOARD-VISUALIZATIONS-SPEC.md` Section 5
- Existing PDF: `src/components/pdf/` (heat-map-pdf, pdf-styles, pdf-header-footer)
- Existing API: `src/app/api/reports/team-heatmap/route.tsx`
- Widget registry: `src/features/dashboard/widget-registry.ts`
- 18 registered widgets across 4 categories

## Tasks

### Task 1: Pre-export modal component (R31-01)
type="auto"
- Create `src/features/dashboard/pdf-export/export-pdf-modal.tsx`
- Checkbox list of active dashboard widgets with select/deselect all
- Orientation toggle (landscape/portrait)
- Cover page checkbox
- Export button triggers generation

### Task 2: SVG snapshot utility (R31-02)
type="auto"
- Create `src/features/dashboard/pdf-export/svg-snapshot.ts`
- Extract SVG from DOM Recharts containers
- Convert to data URI for @react-pdf/renderer Image
- Table-based widgets get react-pdf native rendering instead

### Task 3: Multi-widget PDF layout engine (R31-03)
type="auto"
- Create `src/features/dashboard/pdf-export/dashboard-pdf-document.tsx`
- Each widget rendered as a section with title
- Large widgets (colSpan 12) get full page
- Small widgets (colSpan 4-6) pack 2-3 per page
- Extend pdf-styles with new multi-widget styles

### Task 4: Single-widget export from menu (R31-04)
type="auto"
- Add "Export as PDF" option to widget wrapper
- Direct download, no modal
- Uses same PDF renderer for single widget

### Task 5: Cover page template (R31-05)
type="auto"
- Create `src/features/dashboard/pdf-export/pdf-cover-page.tsx`
- Org name, dashboard title, date range, generated timestamp
- Clean professional layout

### Task 6: API route and wiring
type="auto"
- Create `src/app/api/reports/dashboard/route.tsx`
- Accept widget list, orientation, cover page flag
- Generate multi-widget PDF server-side
- Wire export button into dashboard-layout-engine.tsx

## Verification

- TypeScript compiles without errors
- All new files properly import/export
- Modal renders checkbox list of widgets
- PDF generation handles multiple widget types
