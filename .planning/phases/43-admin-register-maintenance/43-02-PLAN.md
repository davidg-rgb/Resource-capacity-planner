  </behavior>

       - Export a `DependentRowsError extends Error` class with `blockers: Record<string, number>`.
       - Export the four hooks. Use `useQuery` / `useMutation` from `@tanstack/react-query`. Mutation error handler parses `response.json()` and, if `body.error.code === 'DEPENDENT_ROWS_EXIST'`, throws `new DependentRowsError(body.error.details.blockers)`.
       - `queryKey` shape MUST be stable so 43-04's change-log invalidation can target it.
    2. Create `src/hooks/__tests__/use-admin-registers.test.tsx` using `@testing-library/react-hooks` or the repo's existing hook test pattern (grep `renderHook` under existing test files):

       - Mock fetch to return 409 `{ error: { code: 'DEPENDENT_ROWS_EXIST', details: { blockers: { allocations: 3 } } } }` → archive mutation rejects with a DependentRowsError whose `.blockers.allocations === 3`
