    - `type CapacityStatus = 'green' | 'amber' | 'red' | 'gray'`
    - `calculateStatus(sumHours: number, targetHours: number): CapacityStatus`:
      - If targetHours is 0 or sumHours is 0: return `'gray'`
      - ratio = sumHours / targetHours
      - ratio < 0.9: return `'green'`
      - ratio <= 1.0: return `'amber'`
      - ratio > 1.0: return `'red'`
      - These thresholds come from INPUT-05: green (<90%), amber (90-100%), red (>100%), gray (no allocations)
    - `getStatusColor(status: CapacityStatus): string` -- maps to Tailwind-compatible color strings:
      - green -> `'bg-green-500'`
      - amber -> `'bg-amber-500'`
      - red -> `'bg-red-500'`
      - gray -> `'bg-gray-300'`
