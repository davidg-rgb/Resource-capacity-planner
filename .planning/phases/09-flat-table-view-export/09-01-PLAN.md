**allocation.service.ts** -- ADD three new functions (keep all existing functions unchanged):

1. **Helper `buildFlatConditions`** -- Shared between list and count (per Pitfall 6: count query must match data query joins). Takes `orgId: string` and `filters: FlatTableFilters`. Builds an array of Drizzle conditions:
   - Always: `eq(schema.allocations.organizationId, orgId)`
   - If `filters.personId`: `eq(schema.allocations.personId, filters.personId)`
   - If `filters.projectId`: `eq(schema.allocations.projectId, filters.projectId)`
   - If `filters.departmentId`: `eq(schema.people.departmentId, filters.departmentId)`
   - If `filters.monthFrom`: `gte(schema.allocations.month, `${filters.monthFrom}-01`)` (append -01 per Pitfall 1)
   - If `filters.monthTo`: `lte(schema.allocations.month, `${filters.monthTo}-01`)`
   - Returns the conditions array.

2. **Helper `buildFlatBaseQuery`** -- Shared query builder with all 4 JOINs. Returns `db.select({...}).from(schema.allocations).innerJoin(people).innerJoin(departments).innerJoin(projects).leftJoin(programs)`. Select fields:
   - `personName: sql<string>\`${schema.people.firstName} || ' ' || ${schema.people.lastName}\``
   - `departmentName: schema.departments.name`
   - `projectName: schema.projects.name`
   - `programName: schema.programs.name`
   - `month: schema.allocations.month`
   - `hours: schema.allocations.hours`

   Join conditions:
   - `innerJoin(schema.people, eq(schema.allocations.personId, schema.people.id))`
   - `innerJoin(schema.departments, eq(schema.people.departmentId, schema.departments.id))`
   - `innerJoin(schema.projects, eq(schema.allocations.projectId, schema.projects.id))`
   - `leftJoin(schema.programs, eq(schema.projects.programId, schema.programs.id))` (leftJoin because programId is nullable, per Pitfall 5)

3. **`listAllocationsFlat(orgId, filters, pagination: { page: number; pageSize: number })`** -- Returns `FlatTableRow[]`. Uses `buildFlatBaseQuery`, applies `where(and(...buildFlatConditions()))`, adds `orderBy(schema.people.lastName, schema.people.firstName, schema.allocations.month)`, adds `.limit(pagination.pageSize).offset((pagination.page - 1) * pagination.pageSize)`. Map results to normalize month with `normalizeMonth()` and handle null programName (coalesce to `null`).

4. **`countAllocationsFlat(orgId, filters)`** -- Returns `number`. Same JOINs via a select of `sql<number>\`count(*)\``, same `where(and(...buildFlatConditions()))`. Returns `Number(result[0].count)`.

5. **`exportAllocationsFlat(orgId, filters, format: 'xlsx' | 'csv')`** -- Returns `Buffer`. Fetches ALL matching rows (no pagination limit -- use pageSize 100000 per D-11). Builds SheetJS workbook following the pattern from `import.templates.ts`:

   - Column widths: `[{ wch: 25 }, { wch: 15 }, { wch: 25 }, { wch: 20 }, { wch: 10 }, { wch: 8 }]`
   - Sheet name: `'Allocations'`
   - `XLSX.write(wb, { type: 'buffer', bookType: format === 'csv' ? 'csv' : 'xlsx' })`
   - Return `Buffer.from(buf)`

Import additions needed at the top of allocation.service.ts: `gte`, `lte`, `sql` from `drizzle-orm`, `* as XLSX from 'xlsx'`, and the new types `FlatTableFilters`, `FlatTableRow`.

**src/app/api/allocations/flat/route.ts** -- NEW file. GET handler for paginated flat table data (per D-03, D-06, D-07):

```typescript
import { NextRequest, NextResponse } from 'next/server';
import { listAllocationsFlat, countAllocationsFlat } from '@/features/allocations/allocation.service';
import { handleApiError } from '@/lib/api-utils';
import { getTenantId } from '@/lib/auth';
import type { FlatTableFilters } from '@/features/allocations/allocation.types';

export async function GET(request: NextRequest) {
  try {
    const orgId = await getTenantId();
    const params = request.nextUrl.searchParams;

    const filters: FlatTableFilters = {
      personId: params.get('personId') ?? undefined,
      projectId: params.get('projectId') ?? undefined,
      departmentId: params.get('departmentId') ?? undefined,
      monthFrom: params.get('monthFrom') ?? undefined,
      monthTo: params.get('monthTo') ?? undefined,
    };

    const page = Math.max(1, Number(params.get('page') ?? '1'));
    const requestedSize = Number(params.get('pageSize') ?? '50');
    const pageSize = [25, 50, 100].includes(requestedSize) ? requestedSize : 50;

    const [rows, total] = await Promise.all([
      listAllocationsFlat(orgId, filters, { page, pageSize }),
      countAllocationsFlat(orgId, filters),
    ]);

    return NextResponse.json({
      rows,
      pagination: { page, pageSize, total, totalPages: Math.ceil(total / pageSize) },
    });
  } catch (error) {
    return handleApiError(error);
  }
}
```

```typescript
import { NextRequest, NextResponse } from 'next/server';
import { exportAllocationsFlat } from '@/features/allocations/allocation.service';
import { handleApiError } from '@/lib/api-utils';
import { getTenantId } from '@/lib/auth';
import type { FlatTableFilters } from '@/features/allocations/allocation.types';

export async function GET(request: NextRequest) {
  try {
    const orgId = await getTenantId();
    const params = request.nextUrl.searchParams;

    const filters: FlatTableFilters = {
      personId: params.get('personId') ?? undefined,
      projectId: params.get('projectId') ?? undefined,
      departmentId: params.get('departmentId') ?? undefined,
      monthFrom: params.get('monthFrom') ?? undefined,
      monthTo: params.get('monthTo') ?? undefined,
    };

    const format = params.get('format') === 'csv' ? 'csv' : 'xlsx';
    const buffer = await exportAllocationsFlat(orgId, filters, format);

    const date = new Date().toISOString().slice(0, 10);
    const filename = `allocations-${date}.${format}`;
    const contentType = format === 'csv'
      ? 'text/csv; charset=utf-8'
      : 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet';

    return new NextResponse(buffer, {
      headers: {
        'Content-Type': contentType,
        'Content-Disposition': `attachment; filename="${filename}"`,
      },
    });
  } catch (error) {
    return handleApiError(error);
  }
}
```
