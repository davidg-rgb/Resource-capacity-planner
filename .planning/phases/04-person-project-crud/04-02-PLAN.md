export const projects = pgTable('projects', {
  id: uuid('id').defaultRandom().primaryKey(),
  organizationId: uuid('organization_id').notNull().references(() => organizations.id),
  name: varchar('name', { length: 200 }).notNull(),
  programId: uuid('program_id').references(() => programs.id),  // nullable FK
  status: projectStatusEnum('status').default('active').notNull(),  // 'active' | 'planned' | 'archived'
  archivedAt: timestamp('archived_at', { withTimezone: true }),
  createdAt: timestamp('created_at', { withTimezone: true }).defaultNow().notNull(),
  updatedAt: timestamp('updated_at', { withTimezone: true }).defaultNow().notNull().$onUpdate(() => new Date()),
}, (t) => [
  unique('projects_org_name_uniq').on(t.organizationId, t.name),  // UNIQUE name per org
]);

export const projectStatusEnum = pgEnum('project_status', ['active', 'planned', 'archived']);

    Try `createInsertSchema` from 'drizzle-zod' first (same approach as person.schema.ts in Plan 01).
    - `projectCreateSchema`: omit id, organizationId, createdAt, updatedAt, archivedAt. Name: `z.string().min(1).max(200)`. programId: `z.string().uuid().nullish()`. status: `z.enum(['active', 'planned']).default('active')` (do NOT allow 'archived' via create -- archiving is a separate action).
    - `projectUpdateSchema`: `projectCreateSchema.partial()`
    - Export inferred types.

    **Step 3: Create src/features/projects/project.service.ts**
    Import db from '@/db', schema, eq/and/ilike/isNull/ne from 'drizzle-orm', withTenant, error classes.

    Functions:
    - `listProjects(orgId: string, filters: ProjectFilter = {})`: Build query on `schema.projects` with conditions array starting with `eq(schema.projects.organizationId, orgId)`. Default: exclude archived by adding `ne(schema.projects.status, 'archived')` unless `filters.includeArchived` is true. Filter by programId, status, search (ilike on name) if provided. Order by status (active first, then planned), then name.

    - `getProjectById(orgId: string, id: string)`: Query with `and(eq(projects.id, id), eq(projects.organizationId, orgId))`. Throw `new NotFoundError('Project', id)` if no result.

    - `createProject(orgId: string, data: ProjectCreate)`: Use `withTenant(orgId).insertProject({ name: data.name, programId: data.programId ?? null, status: data.status ?? 'active' }).returning()`. Return first element. The unique constraint `projects_org_name_uniq` will throw a Postgres error on duplicate names -- handleApiError in the route will catch error code 23505 and convert to ConflictError.

    - `updateProject(orgId: string, id: string, data: ProjectUpdate)`: Use `withTenant(orgId).updateProject(id, { ...data, updatedAt: new Date() }).returning()`. If empty result, throw `new NotFoundError('Project', id)`.

    - `archiveProject(orgId: string, id: string)`: Use `withTenant(orgId).updateProject(id, { status: 'archived', archivedAt: new Date() }).returning()`. If empty result, throw `new NotFoundError('Project', id)`.

    **Step 4: Create src/app/api/projects/route.ts**
    Follow same pattern as people/route.ts:
    - `GET`: `getTenantId()`, extract searchParams (programId, status, search), call `listProjects(orgId, filters)`, return `{ projects }`.
    - `POST`: `requireRole('planner')`, parse body with `projectCreateSchema`, call `createProject`, return 201.

    **Step 1: Create src/hooks/use-projects.ts**
    'use client' file with TanStack Query hooks:

    - `useProjects(filters?: Record<string, string>)`: useQuery with queryKey `['projects', filters]`, fetches `GET /api/projects?${params}`, returns typed project array.

    - `useCreateProject()`: useMutation POST to `/api/projects` with JSON body. On success, invalidate `['projects']`.

    - `useUpdateProject()`: useMutation PATCH to `/api/projects/${id}` with JSON body. Variable type: `{ id: string; data: ProjectUpdate }`. On success, invalidate `['projects']`.

    - `useArchiveProject()`: useMutation DELETE to `/api/projects/${id}`. On success, invalidate `['projects']`.

    - `usePrograms()`: useQuery queryKey `['programs']`, fetch `GET /api/programs`. Create a minimal read-only API route at `src/app/api/programs/route.ts`: GET only, calls `withTenant(orgId).programs()`, returns `{ programs }`.
