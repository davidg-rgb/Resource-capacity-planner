2. Create `src/features/import/import.service.ts` with two main functions:

**validateImportRows(orgId: string, rows: ImportRow[]): Promise<ValidationResult>**
  - Fetch all people for the org: `const people = await listPeople(orgId, { includeArchived: false })`. Build name lookup: `people.map(p => ({ id: p.id, name: \`${p.firstName} ${p.lastName}\` }))`.
  - Fetch all projects for the org: `const projects = await listProjects(orgId, { includeArchived: false })`. Build name lookup: `projects.map(p => ({ id: p.id, name: p.name }))`.
  - For each ImportRow, create a ValidationRow:
    a. **Person matching:** Call `matchName(row.personName, peopleList)` using `string-similarity`'s `findBestMatch()`. Exact match (case-insensitive): status='exact', matchId set. Score >= 0.95: status='exact' (auto-match). Score >= 0.8: status='fuzzy', include top 3 suggestions sorted by score. Below 0.8: status='unknown'.
    b. **Project matching:** Same logic against projects list.
    c. **Hours validation:** Must be a positive integer 1-999. Non-numeric or out-of-range: add error "Invalid hours: {value}".
    d. **Month validation:** Must match YYYY-MM. Invalid: add error "Invalid month format: {value}".
    e. **Determine row status:** If any errors[] -> 'error'. If personMatch.status='fuzzy' or projectMatch.status='fuzzy' -> 'warning'. If personMatch.status='unknown' or projectMatch.status='unknown' -> 'error' with error "Person not found: {name}" or "Project not found: {name}". Otherwise -> 'ready'.
  - Build summary by counting statuses.
  - Return ValidationResult.

  Internal helper `matchName(importedName: string, existing: Array<{id: string; name: string}>)`:
  - Exact match: `existing.find(e => e.name.toLowerCase() === importedName.toLowerCase())`
  - If no exact, use `findBestMatch(importedName, existing.map(e => e.name))` from `string-similarity`
  - Return `{ status, matchId?, matchName?, score?, suggestions? }` where suggestions are top 3 matches above 0.6 threshold

**executeImport(orgId: string, rows: Array<{personId: string; projectId: string; month: string; hours: number}>): Promise<ImportResult>**
  - Wrap in `db.transaction(async (tx) => { ... })`:
    - For each row, insert into `schema.allocations` with `onConflictDoUpdate` on the 4-column unique constraint `(organizationId, personId, projectId, month)`, setting `hours` and `updatedAt: new Date()`.
    - Track created vs updated count (same createdAt/updatedAt diff < 1s = created).
  - On success: return `{ imported: rows.length, skipped: 0, warnings: [] }`
  - On error: the transaction auto-rolls back. Return `{ imported: 0, skipped: 0, warnings: [], error: err.message }`
  - Import `db` from `@/db`, `schema` from `@/db/schema`, `and, eq` from `drizzle-orm`.
  - Do NOT use the existing batchUpsertAllocations (it has max 100 limit and conflict detection we don't need). Write a fresh transaction loop.

1. Create `src/app/api/import/upload/route.ts`:
```typescript
import { NextRequest, NextResponse } from 'next/server';
import { requireRole } from '@/lib/auth';
import { parseExcelBuffer, autoDetectMappings } from '@/features/import/import.utils';
import { ValidationError } from '@/lib/errors';

export async function POST(request: NextRequest) {
  const { orgId } = await requireRole('planner');

  const formData = await request.formData();
  const file = formData.get('file') as File | null;
  if (!file) {
    return NextResponse.json({ error: 'No file provided' }, { status: 400 });
  }

  // D-15: max 10MB file size
  if (file.size > 10 * 1024 * 1024) {
    return NextResponse.json({ error: 'File exceeds 10MB limit' }, { status: 400 });
  }

  // Validate file extension
  const ext = file.name.split('.').pop()?.toLowerCase();
  if (!ext || !['xlsx', 'xls', 'csv'].includes(ext)) {
    return NextResponse.json({ error: 'Unsupported file type. Use .xlsx, .xls, or .csv' }, { status: 400 });
  }

  try {
    const buffer = Buffer.from(await file.arrayBuffer());
    // D-16: pass optional codepage from query param for manual override
    const codepageParam = request.nextUrl.searchParams.get('codepage');
    const codepage = codepageParam ? Number(codepageParam) : undefined;

    const parsedFile = parseExcelBuffer(buffer, codepage);
    const mappings = autoDetectMappings(parsedFile.headers);

    return NextResponse.json({
      headers: parsedFile.headers,
      sampleRows: parsedFile.sampleRows,
      totalRows: parsedFile.totalRows,
      formatInfo: parsedFile.formatInfo,
      sheetName: parsedFile.sheetName,
      encodingWarning: parsedFile.encodingWarning,
      suggestedMappings: mappings,
    });
  } catch (err) {
    if (err instanceof ValidationError) {
      return NextResponse.json({ error: err.message }, { status: 400 });
    }
    return NextResponse.json({ error: 'Failed to parse file' }, { status: 500 });
  }
}
```

2. Create `src/app/api/import/validate/route.ts`:
```typescript
import { NextRequest, NextResponse } from 'next/server';
import { requireRole } from '@/lib/auth';
import { validateImportRows } from '@/features/import/import.service';
import { validateRequestSchema } from '@/features/import/import.schema';

export async function POST(request: NextRequest) {
  const { orgId } = await requireRole('planner');

  const body = await request.json();
  const parsed = validateRequestSchema.safeParse(body);
  if (!parsed.success) {
    return NextResponse.json({ error: 'Invalid request', details: parsed.error.issues }, { status: 400 });
  }

  const result = await validateImportRows(orgId, parsed.data.rows);
  return NextResponse.json(result);
}
```
