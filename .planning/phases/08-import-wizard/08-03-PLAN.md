1. Create `src/hooks/use-import.ts` with three TanStack Query mutations:

**useUploadFile()** — `useMutation` that:
  - Takes `{ file: File; codepage?: number }` as input
  - Creates FormData, appends file
  - POSTs to `/api/import/upload` (with ?codepage= if provided)
  - Returns the parsed response containing headers, sampleRows, totalRows, formatInfo, sheetName, encodingWarning, suggestedMappings
  - Type the response to match ParsedFile + `{ suggestedMappings: ColumnMapping[] }`

**useValidateRows()** — `useMutation` that:
  - Takes `{ rows: ImportRow[] }` as input
  - POSTs JSON to `/api/import/validate`
  - Returns ValidationResult

**useExecuteImport()** — `useMutation` that:
  - Takes `{ rows: Array<{ rowIndex: number; personId: string; projectId: string; month: string; hours: number }> }` as input
  - POSTs JSON to `/api/import/execute`
  - Returns ImportResult

All mutations follow the existing pattern: check `response.ok`, throw on error, return typed JSON.
