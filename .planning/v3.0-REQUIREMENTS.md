# Requirements: Nordic Capacity v3.0 — Switch from Excel

**Defined:** 2026-03-30
**Core Value:** Make the app self-explanatory so users switch from Excel without needing training or explanation.

## v3.0 Requirements

### Role-Based Landing (Phase 18)

- [x] **UX-01**: After login, planners and viewers are routed to the team load heat map (`/dashboard/team`), not `/input`
- [x] **UX-02**: After login, admins and owners are routed to the KPI dashboard (`/dashboard`)
- [x] **UX-03**: Heat map page includes a prominent "Plan hours" action button that navigates to the person input grid

### Self-Explanatory Navigation (Phase 19)

- [x] **UX-04**: Navigation labels use task-oriented names that describe what the user will find (not technical labels like "Input" or "Data")
- [x] **UX-05**: Each navigation item has a brief description or subtitle clarifying its purpose
- [x] **UX-06**: A first-time user can identify the correct tab for "see my team's load" and "edit someone's hours" without external guidance

### Heat Map as Hero (Phase 20)

- [x] **UX-07**: Heat map sorts overloaded people to the top by default
- [x] **UX-08**: Heat map shows a summary banner with count of overloaded people and the most critical month
- [x] **UX-09**: Person names in the heat map are clickable and navigate to their planning grid
- [x] **UX-10**: Heat map color thresholds are validated to be instantly distinguishable (green/yellow/red)

### Import-to-Value Flow (Phase 21)

- [x] **UX-11**: After successful bulk import, user is redirected to the team heat map with a success banner showing import count
- [x] **UX-12**: Heat map page includes an "Import from Excel" button as a secondary action
- [x] **UX-13**: The path from "upload spreadsheet" to "see team load visualized" requires no more than 3 clicks

### Swedish Localization — Stretch (Phase 22)

- [ ] **UX-14**: i18n framework is configured with Swedish as default locale and English as fallback
- [ ] **UX-15**: All user-facing strings (nav labels, buttons, alerts, form labels, status messages) have Swedish translations

## Out of Scope

| Feature | Reason |
|---------|--------|
| New dashboard widgets | v3.0 is UX polish, not new features |
| Mobile-responsive layout | Desktop is the primary use case for planners |
| Configurable alert thresholds | Deferred — fixed thresholds sufficient for initial clients |
| Email notifications | No client request yet |
| Custom report builder | Over-engineering for 1-2 orgs |
